Para poder usar el programa debes posicionarte en la carpeta Ejercicio1 TSP(Para ejecutar el programa de TSP) o la carpeta Ejercicio2 SubsetSum(Para ejecutar el programa de SubsetSum) y compilar todos los archivos java con el comando "javac *.java"

Después de eso deber usar el comando "java Prueba" en la carpeta "Ejercicio 1 TSP" para probar el programa del ejercicio del agente viajero o usar el comando 2 o el comando "java Prueba" en la carpeta "Ejercicio 2 SubsetSum" para probar el programa de SubsetSum

La clase Vertice emula un vertice en una grafica

La clase Grafica emula una grafica(aqui se encuentran los metodos que emulan el algoritmo de aproximacion para el problema del TSP)

La clase Prueba es el main del programa(Se llama igual para ambos ejercicios)

La clase Arista emula una arista

La clase SubsetSum tiene la parte del esquema de aproximacion

La clases Ejemplares contienen los Ejemplares que se pueden probar 
El programa le mostrara en pantalla los eemplares disponibles para trabajar el programa, y después de eso se debera seleccionar y pulasar el numero de su eleccion
